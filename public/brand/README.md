# ClauseWala. Brand Assets

Drop your logo files here. Expected filenames:

| File | Usage |
|---|---|
| `logo-mark.png` | Square icon (CW mark) — used in Navbar, Footer, favicon |
| `logo-mark.svg` | SVG version of the icon (preferred for web) |
| `logo-full-dark.png` | Full logo (CW mark + "ClauseWala." text) on dark background |
| `logo-full-light.png` | Full logo on light/white background |
| `favicon.ico` | Browser tab icon (replace `app/favicon.ico`) |
| `og-image.png` | Open Graph social preview image (1200×630 recommended) |

All files in `public/brand/` are served at `/brand/<filename>`.
